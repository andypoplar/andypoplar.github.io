1、typeof 是否能正确判断类型？

对于基础类型来说，除了 null 都可以调用typeof显示正确的类型。

但对于引用数据类型，除了函数之外，都会显示”object”。

因此采用typeof判断对象数据类型是不合适的，采用instanceof会更好，instanceof的原理是基于原型链的查询，只要处于原型链中，判断永远为true

A instanceof B

判断 B 的 prototype 属性指向的原型对象(B.prototype)是否在对象 A 的原型链上。
