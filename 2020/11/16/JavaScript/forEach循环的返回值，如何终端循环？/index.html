forEach中return有效果吗？如何中断forEach循环？

Author: AndyPoplar Date: November 16, 2020  8:19:24 Category: JavaScript 阅读数: loading

本文参考神三元的JS灵魂之问(中)做的笔记，主要记录自己不熟悉的地方。

在forEach中用return不会返回，函数会继续执行。

中断方法：

  1. 在需要中断的地方抛出异常。

  2. 官方推荐方法（替换方法）：用every和some替代forEach函数。every在碰到return false的时候，中止循环。some在碰到return true的时候，中止循环
